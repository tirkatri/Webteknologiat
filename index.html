HTML Tutorial

Johdanto

Tervetuloa tutustumaan HTML -kielen perusteisiin!
HTML (Hypertext Markup Language) on Web-kieli, jolla määritetään verkkosivujen sisältöä ja rakennetta.
Sivun rakenne muodostuu HTML-elementeistä, jotka määritetään HTML-tunnisteilla eli -tageilla.
Useimmat tagit tulevat pareittain, ja ne sijoitetaan tekstin elementin sisällön alkuun ja loppuun.
Tutustu kuhunkin osioon tarkemmin w3schools -sivustolla

HTML-sivun rakenne

HTML -dokumentin kirjoittaminen aloitetaan määrittämällä dokumentin alkuun dokumentin tyyppi <!DOCTYPE html>
Sivuston sisältö kirjoitetaan <html></html> tagien sisälle. Itse dokumentti koostuu kahdesta perusosasta: header ja body.
Opiskele lisää 'Basic'

Head

HTML-dokumentin alussa esitetään ensimmäinen pääosa, header <head> Header -osan sisältö </head>
Header osan sisällä ilmoitetaan sivuston sisältöön liittyviä tietoja, jotka vaikuttavat tekstin sisällön toiminnallisuuteen ja muotoiluun (mm. title, metatagit, css ja javascript tiedostojen linkit) Headerin sisältämä teksti ei näy verkkosivulla.
Opiskele lisää 'Head-element'

Body

Header -osan jälkeen esitetään varsinaisen verkkosivun sisältö body
Verkkosivun sisältö rakennetaan erilaisten elementtien sisään.
Elementeille voidaan määrittää luokkia ja tunnisteita, joiden avulla sivustolle voidaan määrittää muotoilu.

Otsikot

Käytössä on yhteensä kuusi eritasoista otsikkoa:

<h1></h1>

Pääotsikko
<h2></h2>

Alaotsikko
...
<h6></h6>
Alin otsikko

Kappaleet

Tekstiosiot kirjoitetaan kappaletagien väliin

<p> Sivuston sisältöä </p>

HTML ei tunnista kirjoitukseen 'Enter'-painikkeella lisättäviä rivivaihtoja.
Rivinvaihdot lisätään yksittäisellä <br>-tagilla.
Opiskele lisää 'Paragraphs'

Kuvat

Sivustolle upotettavat kuvat ilmaistaan kuvan sijaintiin viittaavalla linkillä:
<img src="https://lab.fi/themes/custom/lab/lab-logo.svg" width="200" height="200">
Linkki voi viitata joko paikalliseen tai ulkoiseen sijaintiin
Kuvalle voidaan lisätä atribuutteja määrittämään esim. kuvan kokoa.

Opiskele lisää 'Images'

Listat ja linkit

Listat voivat olla joko numeroituja <ol></ol> tai numeroimattomia <ul><ul>
Kukin listattava asia ilmaistaan <il></il>-tageilla.

Linkit lisätään tagien <a></a>-sisään. Tagille kirjataan määre sen mukaan, missä linkin sisältö sijaitsee.
Tärkeimmät määreet ovat name, joka viittaa sivun sisäiseen sisältöön, sekä href, joka viittaa ulkoiseen sisältöön, ja koostuu kohdedokumentin URL:stä.

  • LAB
    • <li><a href="https://www.lab.fi/fi">LAB</a></li>
  • LUT
    • <li><a href="https:8//www.lut.fi/fi">LUT</a></li>
